## Overview

This project is a fully playable **command-line Sudoku game written in Python**.
The program loads puzzles from external files, validates moves according to Sudoku and Sudoku X rules, and allows the user to interactively solve puzzles using commands.

The application also includes a **hint system, save/load functionality, undo history, and a recursive solver** capable of completing puzzles automatically.

The project was also designed to demonstrate **algorithmic problem sloving, rule validation, file parsing, and state management in Python**.

## Project Features
The program accepts 9 by 9 text files as inputs for puzzles allowing for varying difficulty and near-endless starting possibilities. The project also allows users to save their game to a similar text file that tracks statistics like, moves, time, and hints. With this save feature, users can resume their game in different play sessions. 

The main goal of this project was to simulate typical features found in many games more complicated than sudoku like main menus, save/loading, and user interface.
